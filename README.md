# Car Purchase Prediction 🚗

## 📌 Problem
Businesses need to identify potential customers who are likely to purchase a car in order to optimize marketing strategies and improve conversion rates.

## 💡 Solution
Developed a machine learning classification model to predict whether a customer will purchase a car based on features like Age and Salary.

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## ⚙️ Approach
- Performed data preprocessing and feature scaling (StandardScaler)
- Split dataset into training and testing sets (80/20)
- Trained multiple models:
  - Logistic Regression
  - Support Vector Machine (SVM - Linear Kernel)
  - K-Nearest Neighbors (KNN)
- Evaluated models using Accuracy Score and Confusion Matrix
- Visualized decision boundaries for comparison

## 📊 Results
- Achieved high classification accuracy across models
- Identified key patterns in customer purchase behavior
- Compared model performance to select the best-performing algorithm
